# MyAppDemo

字节工程训练营客户端方向作业

## 核心功能
点击start开始

LoginPageDemo： 实现注册及登录功能，通过Room存储用户信息，登录后进入个人页面

FeedPageDemo： 展示已注册的用户信息，支持下拉刷新，滚动到底时加载更多，长按删卡功能

## 版本更新
### v1.0.0   架构调整(2025/11/30 )
  - **更新了feed流部分的整体架构**，保证单一职责:  
    adapter负责管理UI，拆分出DataManager负责管理数据，拆分出ViewHolder，支持后续插件式添加卡片样式（还没做注册方法）
  - 添加了下拉加载时的loading动画
  
### v1.1.0   优化个人信息页面(2025/11/30 )
  - 优化了修改昵称，现在点界面内其他地方也会结束编辑
  - 添加了头像，可以点击“生成头像”生成随机头像，头像由服务端下发
  - 添加了两个开关，控制是否在Feed流界面展示自己的头像和视频（具体卡片样式还没实现，视频之后计划使用倒计时模拟）
  - 添加了编辑视频时长功能，模拟视频
  
### v1.2.0   实现卡片注册，添加新样式(2025/12/1 )
  - 在FeedPage中添加了单双列切换开关，可选择单列或双列排版
  - **添加了卡片注册功能**，支持插件式扩展，需要新建卡片类和ViewHolder类，然后在activity中注册即可。
  - 加入了一个新的包含头像的卡片布局，可以与原布局混排。
